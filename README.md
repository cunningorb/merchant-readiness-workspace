# Merchant Readiness Workspace

Merchant Readiness Workspace is a Laravel + Vue application that helps ecommerce merchants evaluate the maturity of their returns operations through guided assessments, transparent scoring, and actionable recommendations.

## Tech Stack

- Laravel
- Vue
- Inertia
- Tailwind CSS
- MySQL

## Goals

- Deliver value before sales engagement
- Help merchants identify operational improvements
- Demonstrate modern Laravel architecture and product thinking

## Status

🚧 In active development.
