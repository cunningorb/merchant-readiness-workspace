# Design Approach

## Vision

Create a self-service Merchant Readiness Assessment that provides genuine value before asking a merchant to schedule a sales call.

## Primary Audience

- Prospective merchants
- Sales engineers
- Customer success

## Design Principles

- Outcome first
- Transparent scoring
- Actionable recommendations
- Trust before conversion
- Progressive disclosure

## UX Flow

Landing Page

↓

Assessment Wizard

↓

Score Generation

↓

Recommendations

↓

Shareable Report

## Assessment Sections

- Business
- Catalog
- Return Policy
- Exchanges
- Manual Operations
- Platform

## Results

Show

- Readiness Score
- Score Breakdown
- Top Opportunities
- Recommended Actions
- Capability Mapping

Use clean SaaS dashboards with cards, score rings, progress bars and responsive layouts.

Frontend stack:

- Vue
- Inertia
- Tailwind
